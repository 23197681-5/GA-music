#GA learning to play piano

A genetic algorithm that learn how to play piano.
 <br> using [Chris Woelk piano's  ](https://codepen.io/cawoelk/pen/GJpMdE )

### Runing:

Inside the root folder:

```bash
$ sudo npm install -g ionic cordova
$ npm install
$ ionic serve
```
